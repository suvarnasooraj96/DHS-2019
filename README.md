\# Pharmacology \& Medical Sciences Quiz Web Application



A sleek, modern, responsive interactive quiz application featuring 100 comprehensive questions covering pharmacology, organic chemistry, and medical sciences. Built purely with Vanilla HTML5, CSS3, and JavaScript.



\## 🚀 Features

\- \*\*100 Curated Questions:\*\* In-depth coverage of medical gas cylinders, vitamin nomenclature, antidote formulations, receptor blockers, and more.

\- \*\*Instant Visual Feedback:\*\* Color-coded option buttons (Green for correct, Red for incorrect) upon selection.

\- \*\*Dynamic Rationales:\*\* Reveals detailed medical and chemical explanations instantly after every question attempted.

\- \*\*Real-time Tracking:\*\* Dynamic progress bar and ongoing live question counters.

\- \*\*Performance Report Card:\*\* Aggregated summary dashboard with customizable milestone messages based on final percentages.

\- \*\*Purely Modular Architecture:\*\* Clean separation of concerns with isolated template views (`index.html`), thematic structural designs (`style.css`), and data logic rules (`app.js`).



\## 🛠️ Tech Stack

\- HTML5

\- CSS3 (Variables, Flexbox, Grid layout systems)

\- JavaScript (Vanilla ES6+ DOM Manipulation)
